# Лабораторна робота №6
Масиви

## Завдання
Реалізувати програму, яка для заданого n обчислює суму ряду:
S = 1/2 + 1/3 + 1/4 + ... + 1/(n+1)

## Реалізація
- Програма зчитує число n.
- За допомогою циклу for обчислюється сума дробів.
- Результат виводиться на екран з точністю 6 знаків.
