Лабораторна робота №6. Масиви

1 ВИМОГИ

1.1 Розробник
Студент групи Баршакаєва Дінара Жаксиликовна КН - 925зі

1.2 Загальне завдання
Необхідно обробити послідовність символів (не NTCS-рядок) та виконати заміну символів згідно правил:
a/A → @
o/O → 0
i/I → 1
s/S → $.

1.3 Індивідуальне завдання
Генерація нікнейму на основі заданої послідовності символів.

2 ОПИС ПРОГРАМИ

2.1 Функціональне призначення
Програма читає кількість символів, формує послідовність char та застосовує заміни для формування нікнейму.

2.2 Опис логічної структури
Алгоритм роботи програми:
1. Отримати від користувача число n.
2. Створити масив символів довжиною n.
3. Зчитати n символів по одному.
4. Послідовно перевірити кожний символ та виконати заміну.
5. Вивести модифіковану послідовність символів.

Точка входу:
Функція main(). Її призначення — початок виконання програми. Інших точок входу бути не може.

2.3 Важливі фрагменти програми
Основний цикл, який виконує заміну символів, знаходиться в main().  
Заміну реалізовано через інструкцію switch.

3 ВАРІАНТИ ВИКОРИСТАННЯ

Приклад:
Вхідні дані:
3
a o b

Результат:
@ 0 b

У прикладі символи вводяться по одному, відповідно до вимог щодо обробки послідовності символів.

4 ВИСНОВКИ
Під час виконання роботи були отримані навички роботи з масивами символів, циклічної обробки та умовних інструкцій. Програма коректно виконує заміну символів згідно поставленого завдання.
