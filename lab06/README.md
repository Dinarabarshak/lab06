Лабораторна робота №6
Тема: Масиви

Завдання:
Генерація нікнейму. В заданій послідовності символів потрібно виконати заміну:
a та A → @
o та O → 0
i та I → 1
s та S → $

Увага: послідовність символів не є NTCS-рядком. Символи обробляються як масив char без термінального нуля.

Індивідуальне завдання:
Обробити послідовність символів довжиною n та виконати вказані заміни.

Структура проекту:
lab06/
  Makefile
  README.md
  doc/lab06.txt
  src/main.c

Компіляція:
make

Запуск:
./lab06

Опис рішення:
Реалізація виконує зчитування масиву символів та обробку кожного елемента циклом. Замінені символи виводяться у новій послідовності.

Точка входу:
Функція main() є єдиною точкою входу в програму відповідно до стандарту C.
